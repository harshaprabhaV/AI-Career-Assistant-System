# 🚀 AI Career Assistant System

An AI-powered career guidance platform that analyzes resumes, calculates ATS scores, recommends suitable job roles, suggests skills to learn, and generates mock interview questions using Machine Learning and Natural Language Processing techniques.

## ✨ Features

- 📄 Resume Upload and Text Extraction
- 📊 ATS Score Calculation
- 💼 Job Recommendation System
- 🎯 Skill Gap Analysis and Skill Suggestions
- 🧠 Mock Interview Question Generation
- 🔍 Semantic Job Matching using Sentence Transformers
- ⚡ Real-Time Predictions through REST APIs

## 🔄 Workflow

1. Upload Resume (PDF)
2. Extract Resume Text
3. Generate Resume Embeddings
4. Match Resume with Job Descriptions
5. Calculate ATS Score
6. Recommend Suitable Jobs
7. Suggest Missing Skills
8. Generate Mock Interview Questions
